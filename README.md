# TRACT

Sequential recommendation benchmarking on RecBole. We benchmark **PSWRec** (our model) against GRU4Rec, SASRec, BERT4Rec, FEARec, and WEARec on Amazon Beauty, MovieLens-1M, and Steam, using a fixed data pipeline and evaluation protocol for fair comparison.

See [experiment/benchmarking_experiments.md](experiment/benchmarking_experiments.md) for the full benchmark design.
